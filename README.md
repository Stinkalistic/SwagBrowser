# SwagBrowser
The MOST EPIC web browser ever made. 

**Features**

 - Has support for 14 html tags
 - Not a chromium fork
 - Image support
 - Favicon support
 - No ads
 - Super Epic
 - Can open hyperlinks
 - Can download files
 - Can play sounds
 - Supports bookmarks

# Shortcomings

 -  Does not support javascript
 - Does not support CSS
 - Does not support WASM
 - Does not support captchas
 - Does not support more than 14 html tags
 - Slow to load websites
 - Majority of websites do not work or look ugly
 - Steals all of your data
 - Does not support extensions
 - Overall very shitty


 # Dependencies
 It requires cmu-graphics, requests, subprocess, argparse, and html.parser
